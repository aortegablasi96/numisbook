# `drizzle` — Generated migrations

SQL migrations generated by `drizzle-kit` from `src/db/schema`.

## Rules

- **Generated, not hand-edited.** Change the schema in `src/db/schema`, then
  regenerate.
- Commit migrations to version control.
- `drizzle.config.ts` (top level, created during implementation) points
  `drizzle-kit` at the schema and this output folder.
