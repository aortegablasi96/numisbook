# GitHub Issue Conventions

## Issue Types

The project tracks work using three issue types:

- Epic
- User Story
- Bug

## Labels

story

## Titles

Epic:
[Epic] Coin Images

Story:
[Story] Upload multiple images per coin

Bug:
[Bug] Coin images fail to upload above 4 MB

## Relationships

Epics contain multiple User Stories.

User Stories may reference ADRs and DDRs.

Bugs should reference the Story they affect whenever possible.
